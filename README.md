# 我的个人网站

一个基于 HTML + CSS 搭建的极简个人展示型网站，可直接部署上线。

## 📋 项目简介

这是一个从零搭建的基础个人网站，包含首页、关于我、作品展示、联系方式四个核心模块。
适配电脑/手机端，可直接通过 GitHub Pages 免费上线。

## 🚀 技术栈

- HTML5：页面结构搭建
- CSS3：页面样式美化、响应式适配
- GitHub Pages：免费静态网站托管

## 📁 项目结构
web/
├── My web.html # 主页面
├── style.css # 样式文件
└── README.md # 项目说明文档
plaintext

## ✨ 功能特性

- 清晰的导航栏，支持锚点跳转
- 卡片式内容布局，视觉整洁
- 响应式设计，在手机/电脑上都能正常显示
- 代码结构简单，容易修改和扩展
- 可直接部署上线，无需后端环境

## 🚀 部署上线（GitHub Pages）

1. 进入仓库 → Settings → Pages
2. Source 选择：`main`（或你当前的分支）
3. 目录选择：`/(root)`
4. 点击 Save
5. 等待 1–2 分钟后，通过类似
   `https://zhu1148.github.io/web/`
   访问你的网站
